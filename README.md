imagepreloader
==============

The same file works in both places.

In Node
-------

You can either 

    var Preload = require('imagepreloader');
    var preload = new Preload;

Or,

In your browser
---------------
    
    <script src='index.js'></script>
    <script>
        var Preload = new Preload;
    </script>

Where index.js is the file found in this directory.
